Data file requirements:

  • CSV format
  • Index column is Date
  • There are several different stock prices: Open, Close, High, Low, etc.
    Each price category occupies a column. The webapp by default takes the 2nd column.
  • Make sure the data file at least covers the year 2016 and 2017.
    (The webapp by default takes price history until the end of year 2016 as the training data,
    and uses data starting from the beginning of year 2017 as test data.)
